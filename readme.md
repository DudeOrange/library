# "Library" Postgres database project #---## О проекте ##Учебный проект по созданию базы данных для библиотеки.Схема базы данных![Image text](https://github.com/DudeOrange/library/blob/main/bd.jpg)В проекте реализованы функции на языке sql и plpgsql (с условиями и циклами), использован пользовательский тип данных, массивы. Для работы с базой данных созданы  пользовательских роли, настроены ограничения ролей, настроено логирование действий с базой.## Структура проекта ##---* docker-compose.yaml - Запускает Docker-контейнер для создания базы данных и подключения к ней pgadmin для отправки скриптов.* run\_log\_db.sh - задает настройки логирования, после выполнения скрипта, требуется перезапустить контейнер для применения настроек.* create_all.sql - скрипт для создания таблиц и ролей в базе данных.* functions.sql - функции и их вызовы для заполнения базы.* views.sql - несколько полезных представлений, для отображения текущего состояния базы.
